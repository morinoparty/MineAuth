MineAuth provides a powerful API for external plugins to register custom HTTP endpoints. This guide will help you get started with addon development.

## Getting Started

To develop an addon for MineAuth, you need to:

1. Add MineAuth API as a dependency
2. Obtain the `MineAuthAPI` instance
3. Create handler classes with annotations
4. Register your handlers

Check out the following guides:

- [HTTP API](/docs/developer/addons/http-api) - HTTP API structure and annotations
- [Register Handler](/docs/developer/addons/register-handler) - How to register custom endpoints
