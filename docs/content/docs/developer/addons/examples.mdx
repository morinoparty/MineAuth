## QuickShop-Hikari Addon

A complete example addon implementation is available in the MineAuth repository:

**[morinoparty/MineAuth/addons/quickshop-hikari](https://github.com/morinoparty/MineAuth/tree/master/addons/quickshop-hikari)**

This addon provides HTTP endpoints for QuickShop-Hikari shop data:

| File | Description |
|------|-------------|
| `QuickShopHikariAddon.kt` | Main plugin class |
| `routes/ShopHandler.kt` | HTTP handler with endpoints |
| `data/*.kt` | Data classes for shop information |

### Generated Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/plugins/quickshophikari/shops` | List all shops |
| GET | `/api/v1/plugins/quickshophikari/shops/{id}` | Get shop by ID |
| GET | `/api/v1/plugins/quickshophikari/users/{uuid}/shops` | Get shops by owner |

## Vault Addon

Provides HTTP endpoints for Vault economy operations:

**[morinoparty/MineAuth/addons/vault](https://github.com/morinoparty/MineAuth/tree/master/addons/vault)**

| File | Description |
|------|-------------|
| `VaultAddon.kt` | Main plugin class |
| `routes/VaultHandler.kt` | HTTP handler with endpoints |
| `data/RemittanceData.kt` | Data class for transfer requests |

### Generated Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/plugins/vault-addon/balance/me` | Get own balance |
| POST | `/api/v1/plugins/vault-addon/send` | Send money to another player |

## BetonQuest Addon

Provides HTTP endpoints for BetonQuest quest data:

**[morinoparty/MineAuth/addons/betonquest](https://github.com/morinoparty/MineAuth/tree/master/addons/betonquest)**

| File | Description |
|------|-------------|
| `BetonQuestAddon.kt` | Main plugin class |
| `routes/QuestsHandler.kt` | HTTP handler with endpoints |
| `data/QuestData.kt` | Data classes for quest information |

### Generated Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/plugins/betonquest-addon/quests` | Get available quests |
