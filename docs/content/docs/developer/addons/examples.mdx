## QuickShop-Hikari Addon

A complete example addon implementation is available in the MineAuth repository:

**[morinoparty/MineAuth/addons/quickshop-hikari](https://github.com/morinoparty/MineAuth/tree/master/addons/quickshop-hikari)**

This addon provides HTTP endpoints for QuickShop-Hikari shop data:

| File | Description |
|------|-------------|
| `QuickShopHikariAddon.kt` | Main plugin class |
| `routes/ShopHandler.kt` | HTTP handler with endpoints |
| `data/*.kt` | Data classes for shop information |

### Generated Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/v1/plugins/quickshophikari/shops` | List all shops |
| GET | `/api/v1/plugins/quickshophikari/shops/{id}` | Get shop by ID |
| GET | `/api/v1/plugins/quickshophikari/users/{uuid}/shops` | Get shops by owner |
