MineAuth provides interactive API documentation through OpenAPI 3.1.0 specification and Scalar UI. The OpenAPI specification is dynamically generated by the plugin, including all registered addon endpoints.

## OpenAPI Specification (Scalar)

The REST API is documented using OpenAPI 3.1.0 specification, rendered with [Scalar](https://github.com/scalar/scalar).

Access the interactive API documentation at the following endpoint on your MineAuth server:

| Endpoint | Description |
|----------|-------------|
| `/scalar` | Interactive API documentation UI (Scalar) |
| `/openapi` | OpenAPI 3.1.0 specification in JSON format |

### Available Endpoint Categories

The OpenAPI specification documents the following endpoint categories:

| Category | Description |
|----------|-------------|
| **OAuth2** | Authorization, token, userinfo, and revocation endpoints |
| **Discovery** | OpenID Connect discovery and JWKs endpoints |
| **Commons** | Server information endpoints (players, plugins) |
| **Plugins** | Dynamically registered addon endpoints |

### Using the Raw Specification

You can retrieve the raw OpenAPI specification from your running server:

```bash
curl http://localhost:8080/openapi
```

This JSON can be imported into tools like:
- [Postman](https://www.postman.com/)
- [Insomnia](https://insomnia.rest/)
- [Swagger Editor](https://editor.swagger.io/)
- Code generators for client SDK generation
