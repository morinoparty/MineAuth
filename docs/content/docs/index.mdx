## What is MineAuth?

MineAuth is a Minecraft plugin that provides OAuth2 and OpenID Connect authentication, allowing other plugins to securely access stored data via HTTP.

<Cards>
  <Card title="Download from Modrinth" href="https://modrinth.com/plugin/mineauth" external description="Get the latest release" />
</Cards>

## Features

- **OAuth2 / OpenID Connect**: Standard authentication protocol support
- **Secure API Access**: JWT-based authentication for HTTP endpoints
- **Plugin Integration**: Easy API for addon plugins to register custom endpoints
- **Annotation-based Routing**: Spring-like `@GetMapping`, `@PostMapping` annotations

:::warning[Warning]

Since MineAuth is currently under development, there is a possibility that disruptive changes to specifications may occur due to version upgrades, etc.

:::
